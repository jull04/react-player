# XState player

Мини веб приложение, с использованием паттерна “Конечные автоматы” для управления состоянием интерфейса


## Ссылка на проект: https://jull04.github.io/react-player/

## Установка и запуск проекта:


### Клонировать репозиторий:

```
git clone https://github.com/jull04/react-player.git
```

### Установить зависимости:

```
npm install
```
### Собрать проект:

```
npm run build
```
### Запустить приложение:

```
npm run start
```
